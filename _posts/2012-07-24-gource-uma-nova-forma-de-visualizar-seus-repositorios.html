Gource – Uma nova forma de visualizar seus repositórios!

Algum tempo atrás fui apresentado ao GOURCE e achei fantástico!

$ man gource

"Gource is an OpenGL-based 3D visualisation tool for source control repositories."

O Gource é um aplicativo para visualização gráfica de todo o histórico de versionamento de um projeto. Ele pode ler informações de um repositório Subversion, Bazaar, Mercurial ou Git para criar belas imagens da interação dos usuários com os arquivos/pastas.

Nada mais é do que uma forma visual (vídeo) de você acompanhar os seus repositórios!

 

Vou descrever abaixo como instalar e sair usando no Linux, que é o meu ambiente de trabalho atual.

$ sudo apt-get install gource
$ git clone repositorio.git #Baixe alguns repositórios do github! =)
$ gource repositorio.git

Existem diversos parâmetros de configuração, para conferí-los basta verificar o velho "manuel" do comando ($gource –help).

 

Após conhecer o gource, tive a idéia de sugerir ao time do github para adicionar mais esta feature, =)

Segue abaixo o retorno do Rick, do time do github, após o meu contato:
