20/11/2013

Open Source/Github

Durante o feriado, acessando o meu facebook me deparo com duas reflexões muito massa do Fabio Akita, são elas!

Open Source

"Open source é um esforço pessoal, para resolver um problema local. Não é um emprego. Neste caso concordo com o DHH: se você for pago $50k para fazer open source, de repente você vai preencher o equivalente a $50k de esforço mesmo sem necessidade. Open source desta forma não é meritocrático, é apenas excludente. Por isso também não acredito em remuneração por open source. Open source é para quem acredita em “scratch your own itch”, como originalmente sempre foi, coçar a própria coceira."

Github não é CV

"Sou culpado disso também, tempos atrás eu diria que é importante contribuir open source e ter isso demonstrado em seu profile no Github. Mas faz pelo menos 3 anos que não considero mais Github como um bom aspecto de CV, no máximo um item curioso mas nem de perto decisivo para qualquer contratação. Entrevista, teste, experimentação em projetos reais ainda são as únicas formas de separar o joio do trigo. Continua sendo importante estar no Github (excelente fonte de estudo e de ferramentas), mas é só isso."


Totally agree!
