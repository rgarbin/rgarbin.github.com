Glue – Bom amigo para ajudar a otimizar seu site!

Hoje existem inúmeras ferramentas para geração de sprites, e uma das que eu mais gosto é o "Glue", pelo simples fato de ser um utilitário de linha de comando para gerar sprites, de uma forma muito simples e rápida. Sim, eu gosto do meu bom e velho terminal (tela preta).

Esta ferramenta ajuda o desenvolvedor a agrupar várias imagens do seu site em uma única imagem otimizada (sprite), e depois manipulá-la através do seu CSS. O sprite ajuda a otimizar nossas páginas pois diminui a quantidade de requisições simultâneas ao servidor, muitas vezes tornando o carregamento mais rápido.

Vamos por a mão na massa, abra o seu terminal e vamos instalar o "Glue".

Se você estiver usando o Debian/Ubuntu para instalar é muito simples. Lembrando que para instalar da forma que eu irei exemplificar é necessário que você teja python instalado em seu ambiente.

Hands On

$ sudo apt-get install python-pip (gerenciador de pacotes python)
$ sudo pip install glue

Depois de instalar Glue podemos verificar se tudo ocorreu corretamente chamando o “help”, para obter a lista de todos os argumentos de linha de comando disponíveis:

$ glue --help

Vamos gerar nossos sprites. Crie uma nova pasta (ícones neste exemplo), e adicionar suas imagens. Depois, você pode simplesmente executar o seguinte comando:

$ glue icons sprites

Pronto, o Glue irá criar uma nova pasta chamada sprites com a seguinte estrutura:

sprites
  ├ ─ ─ icons.css
  └ ─ ─ icons.png

A dica deste post não se limita apenas a apresentar um nova ferramenta(Excelente diga-se de passagem) , mas também em mostrar a importância do uso de sprites, que pode ajudar muito nossos sites a serem mais performáticos!
